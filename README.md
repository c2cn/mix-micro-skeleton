## Mix Micro Skeleton

本项目是 MixPHP 微服务开发程序骨架，包含：

> 每个模块中都包含开箱即用的范例代码

- `JsonRpc` 开发 JSON-RPC 微服务
- `SyncInvoke` 执行同步代码

## 开发文档

MixPHP 开发指南：http://doc.mixphp.cn

## 环境要求

* Linux, OS X, WSL
* PHP >= 7.2
* Swoole >= 4.4.4

## 快速开始

推荐使用 [composer](https://www.phpcomposer.com/) 安装。

安装最新版本：

```shell
composer create-project --prefer-dist mix/mix-micro-skeleton mix 2.1.*
```
